# Spring Boot Course

## О проекте

Учебный репозиторий для курса **"Быстрый старт Разработчик Spring Framework (базовый уровень)"**.  
В проекте 10 заданий, каждое выполняется в отдельной ветке с названием `module-<номер задания>-<краткое название>`
